BASIC USAGE
═══════════════════════════════════════════════════════════════════════
fote "your question"               Quick message
fote -i                            Interactive mode (best)
fote -l                            List all models
fote -h                            Help

WITH MODELS
═══════════════════════════════════════════════════════════════════════
fote -m nemotron-ultra "question"  Use best model (550B)
fote -m deepseek-v4 "question"     Use reasoning model
fote -m codestral "write code"     Use code model
fote -m llama-3.3-70b "question"   Use fast model

INTERACTIVE COMMANDS
═══════════════════════════════════════════════════════════════════════
/model <name>                      Change model
/models                            List all models
/clear                             Clear conversation
/system <prompt>                   Set system prompt
/help                              Show help
/exit                              Quit

INTERACTIVE WORKFLOW
═══════════════════════════════════════════════════════════════════════

[Ask coding questions]          Chat normally

[Ask logic questions]           Chat continues
/clear                          Clear history
/exit                           Quit

TIPS & TRICKS
═══════════════════════════════════════════════════════════════════════
• Use interactive mode (-i) for conversations
• Try different models for same question
• Use code models for programming tasks
• Use reasoning models for math/logic
• Switch models mid-conversation with /model
• Clear history with /clear to start fresh
• Set custom personality with /system
